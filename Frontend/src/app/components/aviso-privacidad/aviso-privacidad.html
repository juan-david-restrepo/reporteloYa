Aviso de Privacidad del Chat

Este aviso complementa nuestra Política de Privacidad y describe cómo manejamos la información recopilada a través de nuestro servicio de chat automatizado (Robotransit).

Grabación del Chat

Almacenamiento en la Nube

Este chat se graba y almacena utilizando un servicio en la nube seguro. Los mensajes intercambiados se almacenan de manera cifrada para proteger tu privacidad.

Fecha de Inicio

Cada conversación queda registrada con su fecha y hora de inicio, permitiendo un seguimiento adecuado del historial de tus consultas.

Al iniciar una conversación con Robotransit, aceptas que tus mensajes sean almacenados y procesados para mejorar nuestro servicio y brindarte asistencia.

Datos que Recopilamos

Mensajes del Chat

Todas las preguntas, consultas y respuestas intercambiadas durante la sesión

Marca de Tiempo

Fecha y horaexacta de cada mensaje enviado

Información de Cuenta

Datos asociados a tu cuenta de usuario registrada

Datos de Conexión

Dirección IP y tipo de dispositivo utilizado

Uso de los Datos

Los datos recopilados a través del chat se utilizan para los siguientes propósitos:

Asistencia

Brindarte respuestas personalizadas a tus consultas

Mejora del Servicio

Analizar patrones de uso para optimizar Robotransit

Seguridad

Detectar y prevenir actividades sospechosas o fraudulentas

Historial

Mantener un registro de tus conversaciones para consulta futura

Protección de tus Datos

Encriptación

Todos los mensajes se transmiten y almacenan de forma cifrada

Almacenamiento Seguro

Servidores seguros con medidas de protección avanzadas

Confidencialidad

Acceso restringido solo al personal autorizado

Retención

Datos almacenados durante el tiempo necesario para el servicio
