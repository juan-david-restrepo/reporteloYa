1. Información del Proyecto

Nombre del Proyecto

Repórtelo Yá

Descripción General

"Repórtelo Yá" es una plataforma digital (aplicación web) para que los ciudadanos del Quindío reporten incidentes viales, con énfasis en infracciones como el mal estacionamiento. El sistema permite ingresar datos del incidente —dirección, hora, foto o video del vehículo infringiendo— y enviar la información directamente a la Secretaría de Tránsito.

Responsable del Proyecto

Equipo de desarrollo de Repórtelo Ya

2. Finalidad del Tratamiento de Datos

Objetivo del Tratamiento

En el proyecto "Repórtelo Ya", el objetivo del tratamiento de datos es recoger y usar la información necesaria para que los usuarios puedan reportar situaciones de manera efectiva, facilitar la gestión de esos reportes y mejorar el servicio, siempre procurando un uso responsable de los datos y enfocado en brindar una experiencia segura y útil para las personas.

Uso de la Información

La información recolectada se utiliza para identificar al usuario de manera básica, mantener comunicación sobre el estado de sus reportes y garantizar un uso responsable de la aplicación. Los datos de localización y las descripciones ingresadas permiten ubicar con precisión el lugar del incidente, facilitando así una atención más rápida y efectiva.

3. Tipos de Datos a Recolectar

Datos Personales

  • Número de Cédula de Ciudadanía

Datos No Personales

  • Ubicación en tiempo real (opcional)
  • Descripción del incidente
  • Imágenes o videos (evidencia)
Método de recolección: Formulario

4. Base Legal del Tratamiento

Normatividad Aplicable

Ley de Protección de Datos Personales - Ley 1581 de 2012

Esta ley establece el régimen general de protección de datos personales en Colombia, regulando la recolección, almacenamiento, uso y circulación de datos personales.

Consentimiento Requerido

Ingreso del número de la Cédula de Ciudadanía y ubicación en tiempo real

5. Almacenamiento y Seguridad de los Datos

Ubicación del Almacenamiento

Base de datos en la nube basada en MySQL

Medidas de Seguridad

  • Encriptamiento de la base de datos
  • Verificación de dos pasos (2FA)
  • Conexiones seguras (SSL/TLS)

Tiempo de Conservación

Indefinido

(Sujeto a políticas de retención)

6. Acceso, Actualización y Eliminación de Datos

Procedimiento

Para solicitar modificación o eliminación de tus datos personales, contacta directamente con el correo de soporte de la aplicación.

Responsable de Atenciones

Personal de "Repórtelo Yá" encargado de administrar las solicitudes o problemas manifestados por los usuarios.

7. Transferencia y Compartición de Datos

¿Se comparten datos con terceros?

Sí, únicamente con el personal de Tránsito.

Finalidad de la Transferencia

Validar de manera correcta la identidad de la persona que hace el reporte en caso de fraude o falsificación de información. Evitar reportes falsos y masivos.

Medidas de Protección

Solo el personal de tránsito tendrá acceso a dicha información en caso de inconsistencia. Quedará un registro de consulta con todos los datos del agente que consultó la información.

8. Riesgos y Controles

Posibles Riesgos

  • Filtración de datos del usuario por parte de personal corrupto de Tránsito
  • Ciberataques a la base de datos o aplicación

Medidas de Mitigación

  • Extremar medidas de seguridad en la encriptación de la base de datos
  • Implementar protocolos de seguridad robustos en la aplicación
  • Validación propia del denunciante antes de validar identidad
  • Registro completo de consultas (datos del agente consultante)
